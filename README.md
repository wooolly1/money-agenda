# 💰 أجندة فلوسي

موقع بسيط لتتبّع الميزانية الشهرية ومصاريفك اليومية — يعمل مباشرة في المتصفح بدون أي تثبيت أو خادم، وكل بياناتك محفوظة على جهازك فقط.

## المميزات

- **ميزانية شهرية**: حدّد مبلغ مصروفك للشهر.
- **تسجيل المصاريف اليومية**: أضف كل مصروف (المبلغ + الفئة + التاريخ + ملاحظة) ويُخصم مباشرة من ميزانيتك.
- **كم باقي لك؟**: يحسب المتبقّي فورًا مع شريط تقدّم لنسبة الصرف.
- **ميزانية لكل أسبوع**: يقسم الشهر إلى أسابيع ويوزّع الميزانية عليها، ويحذّرك إذا قربت تتعدّى الحد الأسبوعي أو تعدّيته.
- **وين راحت فلوسك؟**: تفصيل المصاريف حسب الفئة بالنِّسب والمبالغ.
- **سجل المصاريف**: قائمة بكل عملياتك مع إمكانية الحذف.
- يدعم عملات متعددة وواجهة عربية (RTL).

## التشغيل

افتح ملف `index.html` في أي متصفح. لا يحتاج أي إعداد.

> البيانات تُحفظ في `localStorage` الخاص بالمتصفح، فلا تُرسل لأي مكان.

## الملفات

- `index.html` — هيكل الصفحة.
- `styles.css` — التصميم.
- `app.js` — المنطق (الحسابات، الحفظ، التنبيهات).
